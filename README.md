# Yukoni [MERN Stack Project Template]
ReactJS, Redux, Material-ui, Node, ExpressJS, MongoDB, Webpack

## Quickstart

```
  git clone https://github.com/Yukoni/Yukoni.git
  cd your_App_
  npm install
  npm run bundle
  npm start
```

**Note : Please make sure your MongoDB is running. at localhost:27017**
